Avem în stoc peste 50 de modele de amplificatoare pentru toate necesitățile și bugetele.

  • Amplificatoare
  • Pedale de Efecte
  • Sisteme Wireless
  • Echipamente de Înregistrare

Termenul overdrive se referă la un efect sonor care simulează sunetul unui amplificator împins dincolo de limitele sale normale, oferind un ton cald și subtil distorsionat.

    Formulă frecvență notă muzicală

    Formula utilizată pentru calcularea frecvenței unei note este:

    f n 0 = f n 1 * 2 ^ n 12

    fn0 este frecvența notei dorite. fn1 este frecvența unei note de referință (de obicei, nota A4 = 440 Hz). n este numărul de semitonuri dintre nota de referință și nota dorită.
